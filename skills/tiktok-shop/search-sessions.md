## 6. Error Handling
* `45101001`: Internal error. Retry later. Contact support if fail.
* `36009003`: Internal error. Retry request.

## 7. Pitfalls & Best Practices
* **90-Day Limit:** `begin_time_ge` must be within last 90 days. Older data query fails.
* **Signature:** Generate `sign` using correct algorithm. Incorrect sign causes auth failure.

### Pagination Sequence
To retrieve all records:
1. Send initial request without `page_token`.
2. Check response for `data.next_page_token`.
3. If `next_page_token` is present, send new request with `page_token` set to this value.
4. Repeat until `next_page_token` is empty or null.
