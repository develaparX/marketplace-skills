# Guide: Search Creator Affiliate Orders API

## Overview
API gets creator affiliate orders. Returns order ID, product ID. Use to track creator sales, sync order status.

## Endpoint
*   **Method**: `POST`
*   **URL**: `https://open-api.tiktokglobalshop.com/affiliate_creator/202410/orders/search`

## Headers & Auth
*   `content-type`: `application/json`
*   `x-tts-access-token`: Creator access token (from Get Access Token, `user_type = 1`).

Query string must contain:
*   `app_key`: App identifier.
*   `sign`: Request signature.
*   `timestamp`: Unix timestamp (UTC).

## Parameters

### Query Parameters
| Name | Type | Required | Description |

| `app_key` | string | Yes | Unique app key. |

## Error Handling
*   Check `code` field. Non-zero means error.
*   Read `message` for failure reason.
*   Log `request_id` to debug with support.

## Pitfalls & Best Practices
*   **Token Type**: Use creator token (`user_type = 1`). Shop token fails.
*   **Pagination**: Use `page_token` from response to fetch next page.
*   **Time Format**: Use Unix epoch seconds. Milliseconds cause filter failure.
