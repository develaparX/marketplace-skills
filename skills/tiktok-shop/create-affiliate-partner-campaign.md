# Implementation Guide: Create Affiliate Partner Campaign

## 1. Purpose
API creates affiliate campaigns. Use to recruit sellers, set commissions, define registration periods.

## 2. Endpoint
*   **Method:** `POST`
*   **URL:** `https://open-api.tiktokglobalshop.com/affiliate_partner/202405/campaigns`

## 3. Headers & Authentication
*   `content-type`: `application/json`
*   `x-tts-access-token`: Partner access token (required when `user_type` is 3)

## 4. Parameters

### Query Parameters
| Parameter | Type | Required | Description |
| :--- | :--- | :--- | :--- |
| `app_key` | string | Yes | Unique app key. |
| `sign` | string | Yes | Signature generated by algorithm. |
| `timestamp` | number | Yes | Unix timestamp GMT (UTC+00:00). |
| `category_asset_cipher` | string | Yes | Partner identifier. |

### Request Body
| Parameter | Type | Required | Description |
| :--- | :--- | :--- | :--- |
| `name` | string | Yes | Campaign name. Max 50 characters. |
| `description` | string | Yes | Campaign description. Max 1000 characters. |
| `campaign_start_time` | number | Yes | Start time. Unix epoch seconds. |
| `campaign_end_time` | number | Yes | End time. Unix epoch seconds. Max 360 days from start. |
| `registration_start_time` | number | Yes | Seller registration start. Unix epoch seconds. |
| `registration_end_time` | number | Yes | Seller registration end. Unix epoch seconds. |
| `commission_rate` | number | Yes | Commission rate in hundredths of percent (0.01%). Range: 100 to 8000 (1% to 80%). |
| `contact_info` | object | Yes | Partner contact details (email, phone, social). |
| `target_shop_codes` | array\[string\] | No | Allowed seller shop codes. Max 100 items. |
| `target_seller_types` | array\[string\] | No | Eligible seller types. Values: `LOCAL`, `CROSS_BORDER`. |

## 5. Response Fields
| Field | Type | Description |
| :--- | :--- | :--- |
| `code` | number | Status code. 0 means success. |
| `message` | string | Status message or failure reason. |
| `request_id` | string | Request log identifier. |
| `data` | object | Contains `campaign_id` if successful. |

## 6. Error Codes
| Code | Description | Action |
| :--- | :--- | :--- |
| `16032001` | Invalid parameter Creator and Seller, region mismatch. | Check target shop regions. |
| `16032002` | Invalid campaign period. | Check start/end times. Max duration 360 days. |
| `16032003` | Invalid registration period. | Check registration start/end times. |
| `16032004` | Invalid campaign seller scope. | Check target shop codes or seller types. |
| `16032013` | Payment account not yet activated. | Activate partner payment account. |

## 7. Pitfalls & Best Practices
*   **Commission Math:** Multiply desired percentage by 100. Example: 10% commission = `1000`.
*   **Time Limits:** Campaign duration must not exceed 360 days. Use Unix epoch seconds, not milliseconds.
*   **Target Limit:** `target_shop_codes` array limit is 100. Split campaigns if targeting more shops.
