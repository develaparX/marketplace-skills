## Common Pitfalls and Best Practices

**Pitfalls:**

- Using local time for timestamp. Must be UTC.  
- Generating sign incorrectly. Follow API signing guide exactly.  
- Using wrong app_key. Use key specific to your app.  
- Expired access_token. Refresh token before it expires.  
- Not checking `code` in response. Always validate.

**Best Practices:**

- Securely store access_token and app_key. Use environment variables.  
- Cache cipher values if needed, but invalidate appropriately.  
- Log `request_id` for debugging.

- Use HTTPS only.
