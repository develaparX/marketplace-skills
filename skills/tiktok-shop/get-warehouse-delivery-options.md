# Get Warehouse Delivery Options API Guide

## 1. Overview
API gets delivery options for warehouse. Use to check shipping methods before listing product or calculating shipping.

## 2. Endpoint
* **Method**: `GET`
* **URL**: `https://open-api.tiktokglobalshop.com/logistics/202309/warehouses/{warehouse_id}/delivery_options`

## 3. Headers & Auth
* `content-type`: `application/json`
* `x-tts-access-token`: Seller access token (from Get Access Token, user_type = 0).

## 4. Parameters

### Path Parameters
* `warehouse_id` (string, required): Target warehouse ID.

### Query Parameters
* `app_key` (string, required): Unique app key.
* `sign` (string, required): Signature generated by algorithm.
* `timestamp` (number, required): Unix timestamp GMT (UTC+00:00).
* `shop_cipher` (string, required): Shop cipher for target shop.
* `scope` (string, optional): Filter scope. `WAREHOUSE` returns all active options. `PRODUCT` returns options assignable to product.

## 5. Response Structure
Returns JSON object:
* `code` (number): Status code. `0` means success.
* `message` (string): Status message.
* `request_id` (string): Log ID for debugging.
* `data` (object): Container for delivery options.
  * `delivery_options` (array): List of options.
    * `id` (string): Option ID.
    * `name` (string): Option name.
    * `type` (string): Option type (e.g., `STANDARD`).
    * `dimension_limit` (object): Max height, length, width, unit.
    * `weight_limit` (object): Min weight, max weight, unit.
    * `platform` (array): Allowed platforms.

## 6. Error Handling
* **Code `36009003`**: Internal error. Retry request. Contact support if error persists.

## 7. Pitfalls & Best Practices
* **Timestamp expiry**: Generate fresh timestamp for every request. Old timestamp causes signature failure.
* **Scope filter**: Use `PRODUCT` scope when checking options for specific product listing. Use `WAREHOUSE` for general logistics setup.
* **Signature generation**: Include all query parameters in signature calculation. Order parameters alphabetically before signing.
