*   `type` (string, optional): Campaign type. Enum: `MY_CAMPAIGNS`, `GS_SELLING_CAMPAIGNS`, `SELLER_CAMPAIGNS`, `EXCLUSIVE_TIKTOK_SHOP`. Default: `MY_CAMPAIGNS`.
*   `query_type_filter` (string, optional): Filter for `SELLER_CAMPAIGNS` or `EXCLUSIVE_TIKTOK_SHOP`.

## 6. Error Handling

*   `16032001`: Creator and Seller region mismatch. Check region settings.

## 7. Pitfalls & Best Practices
*   **Cipher Key**: Get `category_asset_cipher` first. Call Get Authorized Category Assets API to get cipher.
*   **Signature**: Generate `sign` last. Use exact query parameters and current timestamp to avoid signature mismatch errors.
*   **Pagination**: Use `next_page_token`. Pass token to `page_token` parameter to get next page. Do not hardcode offsets.
