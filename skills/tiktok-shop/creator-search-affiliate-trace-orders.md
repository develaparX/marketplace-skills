## 6. Error Handling
* **Code `36009003`**: Internal error. Retry request. Contact support if error persists.

## 7. Pitfalls & Best Practices
1. **Timestamp Units**: Query parameter `timestamp` uses seconds. Body parameters `time_ge` and `time_lt` use milliseconds. Mixing these causes empty results or errors.
2. **Signature Generation**: Generate `sign` using query parameters only. Do not include request body in signature calculation.
3. **Pagination**: Use `page_token` from response to fetch next page. Do not guess page offsets.
