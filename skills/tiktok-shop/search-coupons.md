# Search Coupons API Implementation Guide

## 1. Purpose
Search coupons. Use to list, filter, paginate shop promotions.

## 2. Endpoint
* **Method**: `POST`
* **URL**: `https://open-api.tiktokglobalshop.com/promotion/202406/coupons/search`

## 3. Headers & Authentication
* `content-type`: `application/json`
* `x-tts-access-token`: Seller access token (from Get Access Token, user_type = 0).

## 4. Parameters

### Query Parameters
| Parameter | Type | Required | Description |
| :--- | :--- | :--- | :--- |
| `shop_cipher` | string | Yes | Shop identifier. |
| `app_key` | string | Yes | Unique app key. |
| `sign` | string | Yes | Signature generated by algorithm. |
| `timestamp` | number | Yes | Unix timestamp GMT (UTC+00:00). |
| `page_token` | string | No | Page cursor. Omit for first page. |
| `page_size` | number | No | Results per page. Range [1, 100]. Default 50. |

### Body Parameters (JSON)
| Parameter | Type | Required | Description |
| :--- | :--- | :--- | :--- |
| `status` | array | No | Coupon status. Values: `NOT_START`, `ONGOING`, `EXPIRED`, `DEACTIVATED`. |
| `title_keyword` | string | No | Coupon title keyword. Length (0, 100]. No fuzzy match. |
| `display_type` | array | No | Display type. Values: `REGULAR`, `LIVE`, `CREATOR_EXCLUSIVE`, `CHAT`, `PROMO_CODE`. |

## 5. Response Structure
* `code` (number): Status code. 0 means success.
* `message` (string): Failure reason.
* `request_id` (string): Request log identifier.
* `data` (object): Return payload.
  * `total_count` (number): Total coupons found.
  * `next_page_token` (string): Cursor for next page.
  * `coupons` (array): List of coupon objects.
    * `id` (string): Coupon ID.
    * `title` (string): Coupon title.
    * `display_type` (string): Display type.
    * `status` (string): Coupon status.
    * `create_time` (number): Creation timestamp.
    * `update_time` (number): Update timestamp.
    * `claim_duration` (object): Start/end times for claiming.
    * `redemption_duration` (object): Start/end times for use.

## 6. Error Handling
* **Code `36009003`**: Internal error. Action: Retry request. If fail persists, contact support.

## 7. Pitfalls & Best Practices
* **Title search**: Exact match only. Fuzzy search not supported.
* **Pagination**: Use `next_page_token` from response as `page_token` in next request.
* **Signature**: Generate `sign` using platform algorithm before sending request. Timestamp must be current.
