# Image Translation API Guide

## 1. Overview
API start image translation. Use when need translate product image text to other language.

## 2. Endpoint
* **Method**: `POST`
* **Path**: `/product/202505/images/translation_tasks`

## 3. Headers & Authentication
* **Scope**: `serller.product.optimize`
* **Headers**:
  * `content-type`: `application/json` (Required)
  * `x-tts-access-token`: Seller access token (Required)

## 4. Parameters

### Query Parameters
| Name | Type | Required | Description |
| :--- | :--- | :--- | :--- |

## 6. Error Handling

| :--- | :--- | :--- |
| `36009003` | Internal error. Please try again... | Retry request. Contact support if fail persist. |
| `12038008` | The image URI is invalid... | Use URI from Upload Product Image API. |
| `12038009` | The sp (truncated) | Check parameter payload. |

## 7. Pitfalls & Best Practices
* **Image Source**: External URLs fail. Upload image first via Upload Product Image API to get valid URI.
* **Time Sync**: Server clock must sync with UTC+00:00. Wrong timestamp cause signature mismatch.
* **Sign Generation**: Generate signature using exact query parameters. Order matters.
