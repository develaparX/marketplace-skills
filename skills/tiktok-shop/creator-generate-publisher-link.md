# Creator Generate Publisher Link API Guide

## 1. Overview
API generate affiliate sharing links for publisher. Use when creator share product via publisher. Material type must be `PRODUCT`.

## 2. Endpoint
* **Method**: `POST`
* **URL**: `https://open-api.tiktokglobalshop.com/affiliate_creator/202504/affiliate_sharing_links/publisher/{publisher_id}/generate_batch`

## 3. Headers & Authentication
* `content-type`: `application/json`
* `x-tts-access-token`: Creator access token (from Get Access Token, `user_type = 1`).

## 4. Parameters

### Path Parameters
* `publisher_id` (string, required): Publisher ID in partner system.

### Query Parameters
* `app_key` (string, required): Unique app key.
* `sign` (string, required): Signature generated by algorithm.
* `timestamp` (number, required): Unix timestamp GMT (UTC+00:00).

### Request Body (JSON)
* `material` (object, required): Target entities.
  * `ids` (array of strings, required): Product IDs.
  * `type` (string, required): Value must be `PRODUCT`.
  * `promotion_campaign_schema` (string, optional): Campaign schema URL.
* `campaign_id` (string, optional): Campaign ID if product from campaign.
* `link_type` (string, optional): Pass `TOKO` for Tokopedia link. Leave empty for TikTok Shop link.

## 5. Response Structure
* `code` (number): Status code. `0` means success.
* `message` (string): Status message.
* `request_id` (string): Request log ID.
* `data` (object): Payload.
  * `sharing_links` (array): Successful links.
    * `material_id` (number): Product ID.
    * `sharing_link` (string): Web link.
    * `deep_link` (string): App deep link.
    * `one_link` (string): Universal link.
  * `failed_materials` (array): Failed items.
    * `material_id` (string): Product ID.
    * `fail_reason` (string): Cause of failure.

## 6. Error Handling
Check `code` in response. If not `0`, request failed. Check `failed_materials` array for item-level errors (e.g., sold out).

## 7. Pitfalls & Best Practices
* **Material Type**: Only `PRODUCT` supported. Request fail if other type used.
* **Tokopedia Agencies**: Must pass `link_type` as `TOKO` to get Tokopedia URL.
* **Partial Success**: API return `code: 0` even if some products fail. Always check `failed_materials` array.
