# Target Collaboration API Implementation Guide

Use API to create target collaboration between seller and creators. Use to invite specific creators to promote specific products.

*   **Method**: `POST`
*   **Path**: `/affiliate_seller/202508/target_collaborations`
*   **Required Scope**: `seller.affiliate_collaboration.write`

## Pitfalls & Best Practices

*   **Limit Limits**: Max 100 products and 50 creator IDs per request. Split payload if limits exceeded.
*   **Time Format**: `end_time` requires Unix epoch timestamp. Pass as string.
*   **Validation**: Check product ownership and creator status before call to avoid partial failures.
