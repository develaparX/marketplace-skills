# TikTok Shop API: Optimized Images

# API Implementation Guide: Optimized Images

## 1. Overview
API optimizes product images. Changes background to white. Use for bulk product image preparation. Process runs asynchronously.

## 2. Endpoint
*   **Method:** `POST`
*   **URL:** `/product/202404/images/optimize`

## 3. Headers & Authentication
*   **Scope Required:** `seller.product.basic`

| Header | Type | Required | Description |
| :--- | :--- | :--- | :--- |
| `content-type` | string | Yes | Must be `application/json` |
| `x-tts-access-token` | string | Yes | Seller access token |

## 4. Parameters

### Query Parameters
| Parameter | Type | Required | Description |
| :--- | :--- | :--- | :--- |
| `app_key` | string | Yes | Unique app key |
| `sign` | string | Yes | Signature generated by algorithm |
| `timestamp` | int | Yes | Unix timestamp GMT (UTC+00:00) |
| `shop_cipher` | string | Yes | Shop identifier |

### Request Body (JSON)
*   `images` (array, required): List of images. Max 200 items.
    *   `uri` (string, required): Image URI from Upload Product Image API.
    *   `optimization_mode` (array of strings, required): Use `["WHITE_BACKGROUND"]`.

```json
{
  "images": [
    {
      "uri": "tos-alisg-i-fh79dxxxx-sg/storage",
      "optimization_mode": ["WHITE_BACKGROUND"]
    }
  ]
}
```

## 6. Error Handling
| Error Code | Message | Resolution |
| :--- | :--- | :--- |
| `12019065` | The image URI is invalid. Please use the URI generated by the Upload Product Image | Upload image first via Upload Product Image API. Use returned URI. |

## 7. Pitfalls & Best Practices
*   **Async Process:** Check `optimize_status` in response. Do not assume instant availability.
*   **Batch Limit:** Max 200 images per request. Split larger batches.
*   **URI Source:** External URLs fail. Use URIs from Upload Product Image API.
